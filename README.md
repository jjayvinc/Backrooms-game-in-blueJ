# The Backrooms: Entity Pursuit

An atmospheric horror-themed game developed in Java. The player must navigate through the endless, procedurally inspired "Backrooms" while being pursued by a hostile entity.

## Features
*   **AI-Powered Pathfinding**: The entity features a custom logic script to track and follow the player's movements.
*   **Asset Integration**: Implements custom PNG textures for an immersive "Backrooms" atmosphere.
*   **Collision Detection**: Robust logic to handle player and entity movement within the generated environment.

## Development Approach
*   **AI-Assisted Coding**: This project was built using a "Technical Architect" approach, where I designed the system architecture and logic while using AI tools to assist in rapid prototyping and implementation.
*   **Rapid Development**: Focused on transforming a conceptual idea into a playable prototype within a short timeframe.

## How to Run
1. Open the project in BlueJ.
2. Ensure all image assets (e.g., `backrooms_waende.png`) are in the project folder.
3. Compile and run the main class.
