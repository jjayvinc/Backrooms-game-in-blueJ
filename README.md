# The Backrooms: Entity Pursuit

An atmospheric horror-themed game developed in Java. The player must navigate through the endless, procedurally inspired "Backrooms" while being pursued by a hostile entity.

## 🛠 Features
*   **AI-Powered Pathfinding**: The entity features custom logic to track and follow the player's movements.
*   **Asset Integration**: Implements custom textures to create an immersive environment.
*   **Collision Detection**: Robust logic to handle movement within the generated walls.

## 🕹 Controls & Gameplay
*   **Movement**: Use the standard movement keys to navigate the halls.
*   **Camera/View**: Use the **Arrow Keys** (Left, Right, Up, Down) to look around and change your perspective.

## ⚠️ Performance Optimization (BlueJ Engine)
*   **Rendering Workaround**: Due to the rendering limitations of the BlueJ internal engine, it is recommended to **downscale/resize the image assets** for a smoother gameplay experience. 
*   **Optimization**: Reducing the image resolution decreases the engine's processing load, significantly increasing the frames per second (FPS).

## 💡 Development Approach
*   **AI-Assisted Coding**: This project was built using a "Technical Architect" approach, where I designed the system architecture and logic while using AI tools to assist in rapid prototyping.
*   **Rapid Development**: Focused on transforming a conceptual idea into a playable prototype within a short timeframe.

## 📖 How to Run
1. Open the project in BlueJ.
2. Ensure all image assets (e.g., `backrooms_waende.png`) are in the project folder.
3. Compile and run the main class.
